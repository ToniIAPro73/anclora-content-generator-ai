# ANCLORA-FEAT-016 — Conversion Landing Test Plan

## Validación funcional
- Verificar que `/` renderiza la landing en lugar de redirigir a `/login`.
- Verificar que el CTA principal lleva a `/login`.
- Verificar que los anchors internos o CTA secundarios funcionan.

## Validación visual
- Revisar hero, proof strip, workflow y CTA final en desktop.
- Revisar legibilidad, jerarquía y respiración visual en móvil.
- Revisar que el logo y la textura de fondo no compiten con el copy.

## Validación técnica
- Ejecutar `npm run lint`.
- Ejecutar `npm run build`.

## Validación de producto
- Confirmar que la página comunica claramente:
  - qué es Anclora
  - para quién es
  - por qué parece premium
  - qué acción debe tomar el usuario
