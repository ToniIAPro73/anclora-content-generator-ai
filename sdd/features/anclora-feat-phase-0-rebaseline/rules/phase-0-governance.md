# Phase 0 Governance

- Toda referencia documental al stack debe distinguir entre auth y persistencia.
- No se puede afirmar aislamiento multi-tenant fuerte mientras `workspaceId` siga viniendo del cliente.
- La documentacion del dashboard debe respetar la gobernanza de layout existente.
- El objetivo de Fase 0 es claridad y baseline, no introducir cambios funcionales amplios.
