# Operational Recommendations Rules

- Las recomendaciones deben ser explicables, no opacas.
- Cada recomendacion necesita un CTA interno y accionable.
- La automatizacion de esta fase solo sugiere; no publica, no reprograma y no modifica contenido sin confirmacion humana.
- El ranking de recomendaciones debe priorizar cuello de botella editorial antes que optimizaciones cosmeticas.
