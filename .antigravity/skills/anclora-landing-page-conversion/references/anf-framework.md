# ANF Framework for Conversion Landing Pages

`ANF` es el marco interno para construir landings de alta conversión sin caer en una homepage SaaS intercambiable.

## A — Authority
La primera pantalla debe hacer que el producto parezca:
- serio
- caro
- específico
- inevitable

Reglas:
- un titular principal
- una promesa clara
- una sola lectura dominante
- un tratamiento visual que respire control

## N — Narrative
La página debe leerse como una secuencia corta:
1. qué cambia con el producto
2. cómo opera el sistema
3. por qué es distinto
4. qué acción debe tomar el usuario

Reglas:
- máximo 4 o 5 bloques
- sin listas largas
- sin taxonomías técnicas
- sin repetir la misma idea con palabras distintas

## F — Frictionless Action
Todo debe converger en una sola acción principal.

Reglas:
- CTA visible en el hero
- CTA repetido solo al final
- secundario solo para reducir fricción
- eliminar navegación o copy que desvíe atención

## Aplicación visual
Para reinterpretar patrones tipo `21st.dev` en clave Anclora:
- usar spotlight cálido, no neón frío
- usar glass y gradients con contención
- usar cards apiladas para tensión visual
- evitar el exceso decorativo y el ruido

La referencia no es copiar componentes, sino absorber:
- claridad compositiva
- profundidad visual
- contraste
- deseo de interacción
